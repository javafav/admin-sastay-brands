User Module Help

Overview

This section provides an overview of the user module and its key features. Only admin users are allowed to create users and manage user accounts, set permissions, and update their profiles.

Creating a User Account

User accounts can only be created by an admin user. If you need an account, please contact your administrator. The admin will create the account and provide you with the necessary login details.

Managing Your Profile

After logging in, you can update your profile information by navigating to the "Profile" section. Here, you can change your profile picture, update contact details, and change your password.

User Permissions

The user module allows administrators to set different permissions for each user. This can include access to specific features, administrative privileges, and content management rights.
